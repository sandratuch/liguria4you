# Liguria4you Website

## Dateien
- `index.html` – Website-Struktur
- `style.css` – Design, Layout, Responsive
- `script.js` – Sprachumschaltung DE / EN / IT
- `assets/header.jpg` – Headerbild ersetzen
- `assets/about.jpg` – Foto der Frau später ersetzen

## Bilder austauschen
1. Neues Headerbild in `assets` legen und `header.jpg` nennen.
2. Neues Portraitfoto in `assets` legen und `about.jpg` nennen.
3. Dateinamen müssen exakt gleich bleiben.

## Kontaktformular
Im HTML steht aktuell:
`https://formspree.io/f/your-form-id`

Das ist ein Platzhalter. Bei Formspree ein Formular erstellen und die URL ersetzen.
